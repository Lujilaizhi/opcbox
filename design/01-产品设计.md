# 产品设计文档

## 「AI超能力鉴定 × OPC工作台」—— WaytoAGI 市集互动站

**版本**：V1.0  
**场景**：7.4 张江 AI 产品市集 · WaytoAGI 社区摊位  
**原则**：手机扫码即用、自助互动为主、3分钟完整体验、成果可带走可传播

---

## 一、产品定义

### 一句话定位
> 一个扫码即玩的互动App，先测你的AI超能力人格，再现场用AI帮你生成一套OPC工作成果——好玩→有用→想加入社区。

### 双引擎设计
| 层级 | 模块 | 作用 | 时长 |
|------|------|------|------|
| **Layer 1（引流层）** | AI超能力鉴定测评 | 吸引路人、社交裂变、降低体验门槛 | 1-2 min |
| **Layer 2（深度层）** | OPC超级个体工作台 | 产出真实价值、绑定社区认知、留存转化 | 2-3 min |

### 核心差异点
- 别家摊位：你站着看摊主演示 → 被动
- 我们摊位：你自己手机扫码玩 → 主动参与
- 最后都能带走在自己手机上，截图就能发朋友圈

---

## 二、用户旅程

```
路人经过摊位
  │
  ├─ 看到立牌/二维码 → "测测你的AI超能力？"（好奇心驱动）
  │
  ├─ Layer 1：扫码进入 → 回答6道题（全单选，30秒答完）
  │     ↓
  │   即时生成「AI超能力人格卡」
  │   - 你的超能力名称（如"内容炼金术师"）
  │   - 一句话描述
  │   - 适合你的AI工具推荐
  │   - 底部："想看看AI能为你做什么？→ 深度体验"
  │     ↓
  │   用户行为：截图发朋友圈 / 点"深度体验"
  │
  ├─ Layer 2：选场景 → 填3个字段 → AI生成成果
  │     4个场景可选：
  │     ① 短视频口播脚本
  │     ② 商业海报文案  
  │     ③ 一人公司创业SOP
  │     ④ 个人IP人设诊断
  │     ↓
  │   生成结果展示 → 可复制/截图带走
  │     底部：WaytoAGI社区介绍 + 入群二维码 + "本内容由WaytoAGI社区共创"
  │
  └─ 离开时：手机里有2张截图（人格卡+AI成果），自然记住WaytoAGI
```

## 三、功能模块详设

### 模块A：AI超能力鉴定测评

**题目设计（6题）**
| 题号 | 问题 | 选项 |
|------|------|------|
| Q1 | 你平时最常和AI对话聊什么？ | 写文案/敲代码/做规划/随便聊 |
| Q2 | 如果AI能帮你做一件事，你最想？ | 赚钱/省时间/学新技能/搞创作 |
| Q3 | 你觉得自己离"一人公司"有多远？ | 已经是/正在路上/想过但没动/来看热闹 |
| Q4 | 你做内容时最头疼什么？ | 没灵感/写不动/不会排版/不知道发哪 |
| Q5 | 你更愿意用AI做？ | 文字创作/图像设计/数据分析/自动化流程 |
| Q6 | 你希望AI更像？ | 靠谱同事/创意搭档/全能助理/学习导师 |

**结果生成（6种人格）**

根据答案组合，映射到预设的6种"AI超能力人格"：

| 人格 | 标签 | 推荐方向 | 推荐Layer 2场景 |
|------|------|----------|----------------|
| 🔥 内容炼金术师 | 适合自媒体、文案 | 文案/脚本 | ①口播脚本 ②海报文案 |
| ⚡ 效率黑客 | 擅长自动化、流程优化 | SOP/工具链 | ③创业SOP |
| 🎨 创意造物主 | AIGC创作达人 | 设计/灵感 | ②海报文案 ④IP诊断 |
| 📊 数据先知 | 理性分析型 | 商业/策略 | ③创业SOP ④IP诊断 |
| 🧭 超级连接者 | 擅长资源整合 | 社区/链接 | ④IP诊断 ③创业SOP |
| 🚀 一人军团 | 全能型OPC | 全链路 | 全部场景 |

### 模块B：OPC工作台（深度生成）

**4个场景×预设Prompt模板**

每个场景下，用户填写3个必填字段：
1. 行业/赛道
2. 核心卖点/产品
3. 目标受众

后端用MiniMax API + 社区精选Prompt模板生成完整内容。

**输出质量标准**
- 短视频脚本：钩子句 + 正文3段 + 转化引导 + 话题标签
- 海报文案：主标题 + 副标题 + 核心卖点3条 + CTA
- 创业SOP：启动步骤5步 + 工具清单 + 避坑提示
- IP诊断：定位分析 + 优势劣势 + 内容策略建议

## 四、信息架构（页面树）

```
/                     → 首页（人格卡 + 开始测评按钮）
/quiz                 → 答题页（6题，每次1题，滑动切换）
/result               → 结果页（超能力人格卡 + 深度体验入口 + 分享引导）
/workbench            → 工作台首页（4个场景卡片选择）
/workbench/generate   → 输入页（3个字段 + 生成按钮）
/workbench/output     → 成果展示页（生成内容 + 复制按钮 + 社区信息）
```

## 五、内容策略

### 社区露出原则
遵循豆包方案的"主产品、辅社区"原则，社区信息仅通过3个触点：

1. **成果页底部**（必须）：轻量标注 + 入群二维码
2. **结果页人格卡底部**（可选）："由WaytoAGI社区共创"
3. **应用内不设独立社区介绍页**，不硬广

### 对外话术
> 这是WaytoAGI社区联合百位OPC创作者打磨的AI体验工具。扫码测测你的AI超能力，还能现场用AI生成一套专属方案——免费带走的。

## 六、现场运营SOP

| 角色 | 人数 | 职责 |
|------|------|------|
| 引导员 | 1人 | 招呼路人、引导扫码、讲解玩法 |
| 操作员 | 1人 | 平板展示App、协助不会操作的用户、回答深度问题 |

**摊位布局**
- C位（70%）：平板展示App + 大二维码立牌（"测测你的AI超能力"）
- 辅助（30%）：WaytoAGI易拉宝 + 社区物料

**应急预案**
- App挂了：平板本地缓存静态展示页，工作人员手动用MiniMax网页版生成
- 没网络：提前截图6种人格卡+4场景样例，iPad本地展示
- 人流过大：纯自助扫码模式，引导员只做招呼不管操作
