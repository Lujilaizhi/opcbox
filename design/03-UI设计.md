# UI设计文档

## 设计系统

### 品牌调性
**科技感 × 轻量化 × 亲和力** — 不像冷冰冰的企业工具，更像一个懂AI的朋友。

### 色彩系统

| Token | 色值 | 用途 |
|-------|------|------|
| `--bg-primary` | `#0f0f1a` | 主背景（深色科技蓝黑） |
| `--bg-card` | `#1a1a2e` | 卡片/面板背景 |
| `--bg-card-hover` | `#222240` | 卡片悬停 |
| `--accent` | `#7c5ce7` | 主强调色（紫蓝渐变起点） |
| `--accent-glow` | `#a78bfa` | 发光/高亮 |
| `--text-primary` | `#f0f0f5` | 主文字 |
| `--text-secondary` | `#a0a0b8` | 次要文字 |
| `--text-muted` | `#6b6b80` | 弱化文字 |
| `--success` | `#34d399` | 完成/正确 |
| `--gradient-hero` | `linear-gradient(135deg, #7c5ce7, #06b6d4)` | Hero区域渐变 |
| `--gradient-card` | `linear-gradient(135deg, #2d2d50, #1a1a35)` | 卡片渐变 |

### 排版

| Token | 值 | 用途 |
|-------|-----|------|
| 标题 | `28px/700` | 页面主标题 |
| 大号文字 | `20px/600` | 卡片标题、人格名 |
| 正文 | `16px/400` | 内容文字 |
| 小字 | `14px/400` | 辅助说明 |
| 字体 | `system-ui, -apple-system, sans-serif` | 全站 |

### 圆角 & 阴影

| Token | 值 |
|-------|-----|
| 卡片圆角 | `16px` |
| 按钮圆角 | `12px` |
| 输入框圆角 | `12px` |
| 卡片阴影 | `0 4px 24px rgba(124,92,231,0.15)` |
| 发光阴影 | `0 0 40px rgba(124,92,231,0.25)` |

### 间距系统（4px基准）
- `xs: 4px` `sm: 8px` `md: 16px` `lg: 24px` `xl: 32px` `2xl: 48px`

## 动效设计

| 动效 | 触发 | 实现 |
|------|------|------|
| Hero光晕浮动 | 首页加载 | CSS animation，背景径向渐变位移 |
| 进度条填充 | 答题推进 | CSS transition width |
| 选项选中反馈 | 点击选项 | scale(0.96) + 边框变色 + 勾选标记 |
| 人格卡翻转 | 结果页展示 | 3D rotateY + 延迟揭示 |
| 标签弹入 | 结果页加载 | 各标签 stagger 50ms fadeInUp |
| 卡片悬浮 | 工作台场景选择 | hover: translateY(-2px) + 阴影增强 |
| 生成加载动画 | 等待API | 三点脉冲 + 文案轮换"正在召唤AI…" |
| 内容渐现 | 成果展示 | 逐行 fadeIn，模仿"AI正在写"效果 |

## 响应式策略

| 断点 | 策略 |
|------|------|
| < 375px | 最小支持，padding收缩 |
| 375-428px | 主设计基准（iPhone/Android主流手机） |
| 428-768px | 卡片宽度max-width 420px居中 |
| > 768px | 平板/展位大屏模式：内容居中max-width 480px |

---

## 无障碍 & 现场特殊考虑

- 所有文字对比度 ≥ 4.5:1（深色背景白字天然满足）
- 选项按钮触控区域 ≥ 44×44px（手指友好）
- 室外林下场景：深色UI不受强光影响，比白底方案更耐看
- 生成失败时：不白屏，展示预制兜底文案 + 重试按钮
